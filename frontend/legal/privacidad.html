Política de Privacidad

Última actualización: 15 de mayo de 2026

En Opita Code nos comprometemos a proteger la privacidad de los usuarios de nuestro sitio web opitacode.com y de nuestros productos digitales, incluyendo Vibe Studio (disponible en vibe.opitacode.com y como aplicación de escritorio). Esta Política de Privacidad describe cómo recopilamos, usamos, almacenamos y protegemos la información personal, en cumplimiento de la Ley 1581 de 2012 y el Decreto 1377 de 2013 de la República de Colombia.

1. Responsable del tratamiento

Juan Nicolás Urrutia Salcedo, propietario de Opita Code, establecimiento de comercio colombiano. Correo: owner@opitacode.com.

2. Datos personales que recopilamos

Podemos recopilar los siguientes datos personales:

  • Datos de cuenta: dirección de correo electrónico proporcionada al registrarse o iniciar sesión mediante enlace mágico (Magic Link) o AWS Cognito.
  • Datos de navegación: dirección IP, tipo de navegador, páginas visitadas, tiempo de visita y origen del tráfico, recopilados de forma anónima a través de herramientas de análisis.
  • Datos de uso del producto: métricas de consumo de tokens de inteligencia artificial, historial de conversaciones con el asistente de IA, preferencias de configuración y plan de suscripción activo.
  • Datos de pago: la información de pago es procesada directamente por Wompi (pasarela de pago colombiana). Opita Code no almacena números de tarjeta de crédito ni datos financieros sensibles.

3. Finalidad del tratamiento

Los datos personales recopilados se utilizan para:

  • Autenticar al usuario y gestionar su sesión en Vibe Studio y demás productos.
  • Proveer los servicios de inteligencia artificial contratados, incluyendo la generación de código, respuestas del asistente y ejecución de herramientas autónomas.
  • Administrar cuotas de uso (tokens diarios/horarios) y planes de suscripción.
  • Responder a consultas y solicitudes de soporte.
  • Mejorar la experiencia de navegación, el contenido del sitio y la calidad de los productos.
  • Cumplir con obligaciones legales y regulatorias.

4. Procesamiento de datos por inteligencia artificial

Vibe Studio utiliza modelos de inteligencia artificial de terceros para proporcionar sus funcionalidades. Al usar el servicio, el usuario reconoce que:

  • Los mensajes enviados al asistente de IA son procesados por proveedores externos como Google (Gemini), DeepSeek, OpenAI y OpenRouter, según el plan y la configuración del usuario.
  • Opita Code no entrena modelos de IA con los datos del usuario. Las conversaciones se envían a los proveedores exclusivamente para generar respuestas en tiempo real.
  • El procesamiento de datos por proveedores de IA se rige por las políticas de privacidad de cada proveedor. Recomendamos consultar sus términos respectivos.
  • Los usuarios del plan gratuito utilizan modelos gestionados por Opita Code (actualmente Google Gemini y DeepSeek). Los usuarios con llaves propias (BYOK) envían datos directamente al proveedor configurado.

5. Llaves de API propias (BYOK — Bring Your Own Key)

Vibe Studio permite a los usuarios autenticados conectar sus propias llaves de API de proveedores de inteligencia artificial. Estas llaves se gestionan de la siguiente manera:

  • Las llaves son cifradas con AES-256 antes de almacenarse en los servidores de AWS (DynamoDB).
  • Las llaves en texto plano nunca se almacenan en el navegador después de la configuración inicial. El frontend solo conserva una referencia cifrada.
  • Opita Code no utiliza las llaves del usuario para ningún propósito distinto a la ejecución de solicitudes de IA iniciadas por el propio usuario.
  • El usuario puede eliminar sus llaves en cualquier momento desde la configuración de Vibe Studio.

6. Base legal del tratamiento

El tratamiento de datos personales se realiza con base en el consentimiento del titular, manifestado al crear una cuenta o al proporcionar voluntariamente su información a través de nuestros canales y productos. El titular puede revocar su consentimiento en cualquier momento.

7. Cookies y autenticación

Opita Code utiliza las siguientes tecnologías de almacenamiento en el navegador:

  • Cookies de autenticación: cookies de sesión bajo el dominio .opitacode.com (establecidas por AWS Cognito) para mantener la sesión del usuario activa entre subdominios.
  • LocalStorage: preferencias de interfaz (tema, posición del panel, configuración de VibeLens) y referencias a proveedores BYOK configurados.
  • Cookies analíticas: utilizadas por herramientas de análisis para medir tráfico de forma anónima.

Para más información, consulte nuestra Política de Cookies.

8. Almacenamiento y seguridad

Los datos se almacenan en infraestructura de Amazon Web Services (AWS) en la región us-east-1. Implementamos las siguientes medidas de seguridad:

  • Cifrado de llaves API con AES-256-CBC.
  • Autenticación mediante tokens JWT firmados (RS256 vía Cognito JWKS o HMAC para enlaces mágicos).
  • Comunicación exclusiva por HTTPS/TLS.
  • Acceso restringido a bases de datos mediante políticas IAM de mínimo privilegio.
  • Registros de uso con TTL automático (los datos de consumo se eliminan automáticamente después de 7 días).

9. Derechos del titular (Ley 1581/2012)

De conformidad con la Ley 1581 de 2012, el titular de los datos tiene los siguientes derechos:

  • Conocer, actualizar y rectificar sus datos personales.
  • Solicitar prueba de la autorización otorgada para el tratamiento.
  • Ser informado sobre el uso que se ha dado a sus datos.
  • Presentar quejas ante la Superintendencia de Industria y Comercio (SIC) por infracciones a la ley.
  • Revocar la autorización y/o solicitar la eliminación de los datos cuando no se requieran para la finalidad acordada.
  • Acceder a sus datos personales de forma gratuita.
  • Solicitar la eliminación de sus llaves API almacenadas y datos de cuenta.

10. Ejercicio de derechos

Para ejercer sus derechos de conocer, actualizar, rectificar, suprimir o revocar el consentimiento, el titular puede enviar una solicitud a: owner@opitacode.com. La solicitud debe incluir nombre, identificación y la descripción clara del derecho que desea ejercer. Responderemos en un plazo máximo de 15 días hábiles.

11. Transferencia de datos a terceros

Opita Code puede compartir datos con los siguientes terceros, exclusivamente para la prestación del servicio:

  • Amazon Web Services (AWS): infraestructura de servidores, autenticación (Cognito), almacenamiento (DynamoDB, S3) y envío de correos (SES).
  • Proveedores de IA: Google (Gemini), DeepSeek, OpenAI y OpenRouter — únicamente para procesar las solicitudes de IA del usuario.
  • Wompi: procesamiento de pagos. Opita Code no almacena datos financieros.
  • Cloudflare: CDN, protección contra ataques DDoS y gestión de DNS.

No vendemos, alquilamos ni compartimos datos personales con terceros para fines de marketing o publicidad.

12. Menores de edad

Los servicios de Opita Code están dirigidos a personas mayores de 14 años. No recopilamos intencionalmente datos de menores de 14 años. Si un padre o tutor detecta que un menor ha proporcionado datos sin autorización, puede solicitar su eliminación escribiendo a owner@opitacode.com.

13. Cambios a esta política

Opita Code se reserva el derecho de modificar esta Política de Privacidad en cualquier momento. Los cambios se publicarán en esta página con la fecha de actualización correspondiente. Se notificará a los usuarios registrados por correo electrónico cuando se realicen cambios sustanciales.

14. Contacto

Para cualquier consulta relacionada con esta Política de Privacidad o el tratamiento de sus datos personales, contáctenos en: owner@opitacode.com.
